HTML: HyperText Markup Language
It is not a programming language that allow us to express logic, like for loop and conditioning. Rather t's indeed a markup language that allow us to specify how a we page should look like.
Tags; opening and closing
html element: between two tags
Attributes: the name followed the opending tag(with in)
Conventionally, the content / values of atttributes are quoted
href: hyper reference (link)
